🚀 CRUD Laravel Project

Sistema completo de CRUD (Create, Read, Update, Delete) desenvolvido em Laravel.

📋 Funcionalidades:

  • ✅ Cadastro de usuários
  • ✅ Listagem de usuários
  • ✅ Visualização de usuários
  • ✅ Edição de usuários
  • ✅ Exclusão de usuários
  • ✅ Validação de formulários

🛠️ Tecnologias:

  • Laravel 11
  • PHP 8+
  • MySQL
  • Bootstrap 5
  • Blade Templates
